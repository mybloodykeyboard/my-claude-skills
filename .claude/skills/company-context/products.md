## DeepQ 시리즈

### 1) DeepQ AI 어시스턴트 (Text)

- **한 줄 정의**: 산업특화 sLLM 기반 텍스트 AI 어시스턴트.
- **핵심 기능**: 질의응답, 채팅, 문서 생성, 문서 요약.
- **타겟**: 산업별 도메인 고객 (건설, 자동차, 석유화학, 제약, 공공 등).
- **차별화**: Vertical RAG + 자체 임베딩 모델 + 도메인 코퍼스. 전문용어 인식·인용·근거 기반 응답.
- **성숙도**: GA. 다양한 산업 적용 사례 보유 (포스코이앤씨, SK캐미칼, 한국자동차연구원, 페루정부, 고양시 등).

### 2) DeepQ MetaGen (Video / VLM)

- **한 줄 정의**: 영상·이미지·음성 데이터 기반 메타데이터·요약 생성 및 의미기반 통합검색.
- **핵심 기능**:
  - 영상 기반 설명문 작성, 영상 요약, 동영상 검색
  - 영상 기반 질의응답 (예: "특고압케이블 근처에 접근한 사람 있어?")
  - 영상 메타데이터 자동 생성
  - 의미기반·사물기반·사건기반·분류기반·메타데이터 검색
  - 시나리오 기반 영상탐지, 영상 위험 감지
  - 다국어 지원 (영·중·일), 개인정보 익명화
- **타겟**: 국가기록원 등 공공 (PoC 중), 스마트 안전·교통·CCTV·건설·제조 현장.
- **성숙도**: 일부 GA + 개발 중.

### 3) DeepQ Solution Suite (ML/DL 모델 묶음)

DeepQ Solution Suite v1.0 — **GS인증 1등급 취득**.

구성 모델:
- **DeepQ Anomaly Detection**: 비지도 오토인코더 기반 이상징후 탐지 (정신건강, 비점오염원, 영상 기반, 하수처리 단계, 반려식물 등).
- **DeepQ Recommendation Engine**: 추천 알고리즘.
- **DeepQ Prediction Engine**: 예측 모형 (수위, 수량, 반려식물 상태 등).
- **DeepQ Document Summary**: 대규모 문서 요약.

### 4) DeepQ Chat (챗봇 솔루션)

- **한 줄 정의**: 다양한 LLM 기반 챗봇 솔루션 (조달청 벤처나라 등록).
- **핵심 기능**: 챗봇 인터페이스, RAG, 임베딩, Vector DB / Graph DB, Fine-Tuning, 외부 메신저 연계.
- **Foundation Model**: Llama 3, QWEN 2.5, ChatGPT, Gemini.
- **도메인 벡터 DB 보유**: 건설 품질, 자동차산업, 정보통신기술, 정보보호 법령, 시니어 돌봄, 청소년 마음건강.

## PROP — 제안서·문서 생성 협업툴

- **한 줄 정의**: LLM + Vector DB로 제안서·문서를 생성·편집하는 협업 SaaS.
- **핵심 기능**:
  - 사내 축적 참고자료·노하우 검색
  - 빈 제안서 자동 채움
  - 양질의 문서작업 효율화
- **타겟**: B2C, B2B, B2G + 커스터마이징.
- **기술 스택**: GPT 4o / mini, Vector DB, Langchain agents.
- **사례**: 한국자동차연구원 데이터플랫폼 탑재 (Customized PROP).
- **성숙도**: 2024년 오픈 GA.

## 수직 솔루션 (도메인 특화 제품)

### 마음닥 — AI 기반 정신건강 모니터링

- **한 줄 정의**: 핸드폰 Digital Life Log 기반 정신건강 모니터링·이상징후 탐지.
- **핵심 데이터**: 화면 터치 수, 이동거리·경로, 수면시간 등.
- **타겟**: B2G2C (공사공단, 지자체, 단체). 시니어·청소년·1인 가구.
- **차별화**: 정신건강 이상징후 탐지 **특허 등록**. 시니어 중심 아파트 단지 시범 운영 중.
- **활용**: 고독사 위험군 발굴, 복지 사각지대 연계, 사회관계망 확대.

### 스마트 다이닝 시스템

- **한 줄 정의**: 대형 급식시설 전 공정(전처리·조리·배식·세척) AI 위생·안전 모니터링.
- **핵심 기술**: YOLO v9 기반 객체 탐지 (모자·마스크·장갑 착용, 안전거리, 화재).
- **타겟**: B2B / B2G. 정신병원, 요양병원, 공공기관, 초중고.
- **부가 기능**: IoT 센서 (화재·가스·유해물질), 식자재 추천·공동구매, 출퇴근 얼굴 인식.

### 이미지·영상 기반 이상징후 탐지

- **한 줄 정의**: CCTV 영상 기반 사회·경제적 파장 큰 이상행동 사전 탐지.
- **사례**: 주거지 침입·도난, 주거지 인근 대인 범죄, 어린이 보호구역 아동 학대, 지하철 역내 이상행동.
- **기술**: MMACTION PoseC3D — 좌표값 기반 Inference 적합.

### WiseRun — 우승마 예측 서비스

- 검빛 경마에 AI 우승마 예측 서비스 제공 (2021~).
- B2B SaaS 형태.

## 공공·행정 신규 라인업 (2026 Proposal 기준)

### AI 정책 보좌관 — 지자체장 전용 행정 인텔리전스 플랫폼

출처: `AI정책보좌관_V5_랩큐.pdf` (Proposal 2026).

- **한 줄 정의**: 지자체장의 의사결정을 지원하는 LLM 기반 일일 행정 인텔리전스 플랫폼.
- **핵심 가치 3축**:
  - **Daily Intelligence** — 매일 아침 30초 AI 브리핑 (모바일·집무실 듀얼).
  - **Real-time Insight** — 민원 추이·불만 지수 실시간 시각화.
  - **Direct Communication** — 구민과 직접 소통 채널 (카톡 기반 Push·만족도 피드백·24시간 챗봇).
- **핵심 기능**:
  - 민원 카테고리별 실시간 분류, 지역별 히트맵, 주간 추이, 반복 민원 TOP5.
  - AI 감정 분석·불만 강도 지수.
  - 타깃 맞춤 메시지 세그먼트 (지역·연령·관심사별).
  - 시민 만족도 실시간 피드백.
- **아키텍처 3 Layer**:
  - 서비스 — 지자체장 대시보드, 실무진 보고서, 시민 Push, AI 챗봇.
  - AI 분석 — LLM 민원 요약/분류, 감정 분석, 이슈 클러스터링·트렌드 예측, 지역 매핑, 비식별화.
  - 수집 — 콜센터(STT), 홈페이지·SNS, 모바일 앱, 행정 내부 시스템(새올 등), IoT 센서.
- **타겟**: 지자체장 (구청장·시장·군수). 노원구·중랑구·대전 서구·대전 유성구·완주군·용인시 맞춤 시나리오 보유.
- **도입 단계**: 현황 분석(2주) → 파일럿 운영(4주) → 정식 운영(지속).
- **차별점**: "단순한 업무 효율화를 넘어 데이터 기반 행정 혁신". 24x365, 데이터 기반 객관 정책 근거.

### 차세대 지능형 LLM 행정/민원 서비스 솔루션

출처: `차세대 지능형LLM 서비스 솔루션_FPGA결과 포함_260401.pptx`.

- **한 줄 정의**: 국내 최초 지능형 LLM 기반 민원 서비스. Hybrid LLM + 고도화 RAG + RPU(FPGA 가속) 결합.
- **4대 핵심 기술**:
  1. **하이브리드 LLM** — Rule-Base + LLM. 자체 보고 답변 정확도 99%.
  2. **고도화 RAG** — KoNLPy 청킹 + BGE-M3 임베딩 + 하이브리드 검색(Semantic+Keyword) + HyperCLOVA X.
  3. **AI 킬러 서비스** — Fast-Track 핫라인, 정책 레이더, 지능형 이슈 감지·알림, KPI 협업.
  4. **보이스 AI 상담 어시스턴트** — 실시간 STT, AI 답변 가이드, TTS 자동 안내, CTI/IVR 통합.
- **독보적 기술 RPU** (상세는 `differentiation.md`):
  - FPGA 기반 HW 가속 Cross-Encoder 리랭킹.
  - 동시 접속 10명 시 GPU 대비 8.7배, CPU 대비 9.2배 빠름.
  - 전력 70% 절감, 운영비 65% 절감.
- **검증 사례**: 고양시 LLM 민원 서비스 (참조: `track-record.md`).
