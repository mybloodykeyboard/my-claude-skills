## 1) 회사 정체성 — 온톨로지 기반 도메인 특화 sLLM

- "온톨로지 기반 도메인 특화 LLM/LMM 솔루션 선도 기업"이 자기 정의.
- 단순 데이터 처리에서 벗어나 **도메인 지식의 구조화 → 정확한 의미 이해 → AI 신뢰성 확보**를 전략 축으로 함.
- 온톨로지가 주는 5가지 효과:
  1. 정확한 의미 이해 (Semantic Precision) — 문맥기반 응답.
  2. 할루시네이션 방지·신뢰성 — 지식그래프 기반 사실 기반 RAG.
  3. 도메인 특화 대응력 — 건설·의료·법률 등 고난도 도메인의 전문용어·문맥 이해.
  4. 프롬프트 엔지니어링 정확도 — 온톨로지 참조 시멘틱 프롬프트 자동 생성.
  5. 설명가능성 (XAI) — 판단 근거를 개념 수준에서 설명.
- **핵심가치 3축**: 도메인 정밀성 / 지식 구조화 / 실행 중심 혁신.

## 2) 기술 차별화 포인트

### Vertical RAG + RPU (Retrieval Process Unit) — **핵심 차별 기술**

- 자체 명명한 핵심 아키텍처. "범용 LLM의 한계와 Vertical LLM의 필요성"에 대응하는 랩큐 솔루션의 두 기둥.
- 범용 RAG가 전문용어 인식 실패하는 문제(예: PHC파일)를 자체 임베딩 모델 + 도메인 코퍼스로 해결.

**RPU 상세 (Cross-Encoder with HW Acceleration)** — 출처: `차세대 지능형LLM 서비스 솔루션_FPGA결과 포함_260401.pptx`. **PoC 통합 테스트 완료, 개발 완료**.

파이프라인:
1. **1차 벡터 검색** (Bi-Encoder, 속도 중심) — 후보 문서 검색.
2. **정밀 리랭킹** (Advanced Cross-Encoder, 정확도 최상) — 질문-문서 쌍 심층 분석으로 연관성 재평가.
3. **HW 가속 처리** — FPGA 기반 가속기(Alveo U55C)로 Cross-Encoder 고연산 부하 전용 처리.
4. **최적 결과** — High Accuracy + Low Latency 동시 달성.

양자화: INT8/FP16/FP32 모드. **INT8 양자화 시 FP32 대비 ~99.0% 정확도 유지**, 지연 0.25배, 전력 효율 최상.

**벤치마크 (검증된 정량 성과)**:
- 단일 사용자(Batch=1): GPU/FPGA/CPU 지연 시간 근소 — 전체 파이프라인의 지배적 항은 LLM 추론.
- **동시 접속 5명 (Top-k=15)**: FPGA가 GPU 대비 **7.6배** 빠름.
- **동시 접속 10명 (Top-k=20)**: FPGA가 GPU 대비 **8.7배** 빠름. CPU 대비 **최대 9.2배** 낮은 지연.
- **전력 소비 약 70% 절감** (GPU 대비), 연간 전력·냉각 비용 65% 절감.
- 3년 TCO 운영 비용 **65% 절감**.

**권장 아키텍처**: Top-k 10~20 구간에서 **FPGA(리랭킹) + GPU(LLM) 하이브리드** 구성.

> 이는 단순 RAG가 아닌 "환각 억제 + 속도 혁신"을 동시 달성한 자체 HW-SW 결합 기술. 동시 접속이 많은 공공 서비스(민원 챗봇 등)에서 결정적 차별화.

### 하이브리드 LLM (규칙 + 생성형)

- 정형화 업무는 **Rule-Base**, 자연어 이해는 **LLM** — 정확도와 유연성 동시 확보.
- 자체 보고된 **답변 정확도 99%**.
- 멀티턴 맥락 유지 (지칭어 해소: "그것", "저기"를 이전 대화와 연결).
- 정책 기반 답변 제어: 민감 주제·비속어 필터링, 사용자 권한별 답변 차별화 (Safety Filter).

### 고도화 RAG 아키텍처 (한국어 최적화)

- **데이터 전처리**: KoNLPy 청킹 — 단락/표/리스트 인식 기반 의미 단위 최적 분절.
- **벡터 임베딩**: **BGE-M3** 모델 — 한국어 뉘앙스 포착, 다국어/긴 문맥 지원.
- **하이브리드 검색**: Semantic + Keyword 결합 — 정확도/재현율 동시 확보.
- **답변 생성**: **HyperCLOVA X** — 한국어 최적, 출처 근거 제시 포함.
- 효과: 할루시네이션 최소화 (Fact Grounding), 근거 문서 원문 링크 제공, 데이터 최신성 실시간 동기화.

### 자체 임베딩 모델 + 도메인 코퍼스

- **건설지식 코퍼스** 등 도메인별 코퍼스 보유.
- **도메인 벡터 DB**: 건설 품질, 자동차산업, 정보통신기술, 정보보호 법령, 시니어 돌봄, 청소년 마음건강 등 6+ 영역.
- 비표준 사용자 입력을 표준 분류 체계로 LLM 기반 자동 변환 (공종 분류 표준화 등).

### Agentic 구조 + Function Calling

- 공정·품질·원가 등 **Multi-Agent** 구성.
- Semantic Search, SQL Search, 시각화 등 다양한 Function의 동적 오케스트레이션.
- 이종 데이터베이스(Vector / Relational) 간 동적 쿼리 라우팅 구현.
- 기능·데이터 무한 확장 가능한 시스템.

### 멀티모달 RAG

- 도면·이미지 등 비텍스트 자산까지 처리하는 크로스모달 검색 (예: SK케미칼 공장 설비 도면).
- VLM(Vision-Language Model) 기반 영상 해석 서비스 개발 중.

### 자체 RAG 성능 검증 체계

- 자체 RAG 성능 검증 지표 (Top-N Accuracy).
- 자체 Test 데이터셋 구축.
- 생성형 응답 평가 방법 부재 문제에 대한 자체 해결.

### Fine-tuning 다양화

- LoRA fine-tuning (페루 행정 챗봇 등).
- 다양한 Foundation Model 활용: GPT 4o, Llama 3, QWEN 2.5, Gemini, DeepSeek, Solar, Mistral, KoBERT.

### 보이스 AI 상담 어시스턴트

- **실시간 STT (Speech-to-Text)** — 통화 내용 실시간 변환 후 상담 화면 즉시 표출.
- **AI 에이전트 어시스트** — 대화 맥락 실시간 분석, 관련 법령·지침·답변 스크립트 즉시 추천.
- **자연스러운 TTS** — 단순 반복 안내 업무를 고품질 AI 음성이 대신.
- **CTI/IVR 통합** — 기존 전화 시스템 완벽 연동, 개인정보 마스킹 보안 준수.
- 효과: 상담 시간 30% 단축, 상담 품질 표준화, 감정 노동 완화.

### 비지도 이상징후 탐지

- 오토인코더(AutoEncoder) 기반 — 산업별 특화 모델 + 베이스라인 모델 확보.
- 시계열 LSTM 적용 (디지털자산 거래, 보안 이상징후).
- MMACTION PoseC3D 기반 좌표값 영상 이상행동 탐지.
